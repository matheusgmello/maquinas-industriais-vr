# Realidade Virtual - Torno Mecânico

Repositório de documentação e recuperação técnica dos arquivos do projeto de realidade virtual envolvendo um torno mecânico.

## Documentação

A documentação foi criada com MkDocs.

```powershell
.\.venv\Scripts\mkdocs.exe serve
```

Depois acesse:

```text
http://127.0.0.1:8000
```
