# Aprendizado de máquinas industriais utilizando Realidade Virtual (RV)

Esta documentação registra o estado atual dos arquivos encontrados no drive do projeto de realidade virtual associado a máquinas industriais como o torno mecânico.

O objetivo inicial é organizar o que existe, explicar como abrir cada parte e deixar um caminho claro para transformar o material em uma experiência interativa. Também busca facilitar o caminho para próximos bolsistas.

## Resumo do estado atual

- Existe um modelo 3D do torno em formato `.fbx`.
- Foi criada uma cena Unity chamada `TornoMecanico` para visualização do modelo.
- O projeto Unity precisou de recuperação de pacotes para abrir corretamente.
- Não foi encontrada uma cena funcional pronta do torno com interações, animações ou lógica de simulação.
- Existe uma máquina virtual `.ova`, provavelmente usada para padronizar ambiente ou armazenar arquivos de trabalho.

## Como usar esta documentação

Leia primeiro o [Inventário](inventario.md), depois a página [Unity](unity.md) para entender como abrir a cena atual. A página [Roadmap](roadmap.md) sugere próximos passos para evoluir o material.
