# Aprendizado de máquinas industriais utilizando Realidade Virtual (RV)

Projeto educacional de Realidade Virtual para aprendizado de máquinas industriais. O foco atual é o **torno mecânico**. O objetivo de longo prazo é uma fábrica virtual com múltiplas máquinas.

## Estado atual

- Modelo 3D do torno (`.fbx`) importado no Unity com hierarquia completa — cada peça é animável.
- Demo automática implementada: mandril, castanhas, cabeçote, carro longitudinal, transversal, manivela e torre.
- Próximos passos: colisores, XR Origin e interação VR.

## Como usar esta documentação

Consulte a página [Unity](unity.md) para entender como abrir a cena. O [Roadmap](roadmap.md) lista o que está feito e o que vem a seguir.
